Formulae for calculations

You should use the following formulae for the calculations:

Shape

Area

Volume

Circle/sphere

(r = radius)

A = Πr2

V = 4/3Πr3

Triangle/Pyramid

(b = base, h = height, l = length, w = width)

A = 1/2bh

Rectangle/Box

(l = length, h = height, b = width)

A = lh

V = lbh

Pentagon

(a = side length, h = height)

Octagon

(a and s = side length, h = height)
